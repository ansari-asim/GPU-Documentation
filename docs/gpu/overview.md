# GPU Overview

Complete guide for flashing Jetson devices with BSP packages, setting up dGPU Ubuntu environments, and transferring the OS to SSD/SD card.

## Available Platforms

### Jetson Devices

NVIDIA Jetson is a series of embedded AI computing modules designed for edge AI applications.

**Supported Jetson Models:**
- Jetson Nano
- Jetson NX
- Jetson TX2NX
- Developer Kit
- Eagletech-101 (Custom)
- Tacodi

### dGPU (Discrete Graphics Processing Unit)

Desktop and server GPUs for high-performance AI inference and training.

**Supported Configurations:**
- Ubuntu 20.04 Dual Boot
- Ubuntu 22.04 Dual Boot

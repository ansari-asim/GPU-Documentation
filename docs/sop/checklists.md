# SOP Checklists

## Pre-Installation Checklist

- [ ] Workspace is clean and dust-free
- [ ] All tools and materials are available
- [ ] System is powered off and unplugged
- [ ] Static wrist strap is on (ESD protection)
- [ ] All cables are labeled for identification
- [ ] Documentation for GPU specifications is available

---

## Installation Checklist

- [ ] Case is opened safely without damage
- [ ] PCIe slot is identified and clean
- [ ] Expansion slot covers are removed
- [ ] GPU connectors are aligned properly
- [ ] GPU is inserted with audible click confirmation
- [ ] GPU bracket is secured with all screws
- [ ] Power cables are connected firmly
- [ ] All connections are double-checked
- [ ] Case is closed and secured
- [ ] System powers up without errors

---

## Post-Installation Verification

- [ ] BIOS recognizes GPU
- [ ] Drivers install without errors
- [ ] GPU temperatures are normal (under 80°C at idle)
- [ ] Power supply can handle GPU TDP
- [ ] Cooling is adequate (no thermal throttling)
- [ ] No artifacts or instability in test runs

---

## Quarterly Maintenance Checklist

- [ ] Cabinet exterior and interior are clean
- [ ] All dust is removed from internals
- [ ] RAM modules are clean and properly seated
- [ ] GPU contacts are clean
- [ ] All fans are spinning correctly
- [ ] Fan nets are free of blockages
- [ ] Power connectors are secure
- [ ] Rack fittings are tight
- [ ] System temperatures are normal
- [ ] No error messages in logs

---

## Pre-Deployment Checklist

- [ ] Hardware has passed all POST tests
- [ ] Temperatures are stable under load
- [ ] Network connectivity is confirmed
- [ ] All required SDKs are installed
- [ ] Sample applications run successfully
- [ ] monitoring scripts are configured and running
- [ ] Backup systems are in place
- [ ] Escalation contacts are documented
